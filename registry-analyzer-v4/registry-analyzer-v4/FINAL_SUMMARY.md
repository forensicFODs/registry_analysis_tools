## 🌟 v4.0 핵심 기능

### 1. 🤖 AI 기반 포렌식 분석 (완전 통합)

#### ✅ 단일 하이브 AI 분석
- API Key 입력 → 단일 파일 분석 → AI 결과 자동 생성

#### ✅ **Multi-Hive AI 통합 분석** ⭐ NEW!
- API Key 입력 → Multi-Hive 분석 → **AI가 모든 하이브를 통합 분석**
- 7개 하이브의 상관관계, 타임라인, 아티팩트를 AI가 종합 분석
- **상관관계 ≠ AI 분석** (명확한 구분)
  - **상관관계**: 규칙 기반 패턴 매칭 (7가지 패턴)
  - **AI 분석**: 머신러닝 기반 종합 판단 (의심 활동, 권장사항)

#### AI 분석 4가지 섹션
1. **📊 Summary** - 전체 분석 요약
2. **⚠️ Suspicious Activities** - 의심스러운 활동
3. **⏱️ Timeline** - AI 생성 타임라인
4. **💡 Recommendations** - 보안 권장사항

## 💡 상관관계 vs AI 분석 (명확한 구분)

### 📊 상관관계 분석 (Section 2)
- **방식**: 규칙 기반 패턴 매칭
- **목적**: 하이브 간 데이터 연결 찾기
- **예시**:
  - "chrome.exe가 ShimCache와 Amcache에 모두 존재"
  - "USB 장치 A가 SYSTEM과 SOFTWARE에서 발견"
- **특징**: 
  - ✅ 빠름
  - ✅ 정확함 (100% 규칙 기반)
  - ❌ 해석 없음 (단순 매칭)

### 🤖 AI 분석 (Section 4)
- **방식**: 머신러닝 기반 종합 판단
- **목적**: 포렌식 인사이트 제공
- **예시**:
  - "chrome.exe가 비정상 시간(새벽 3시)에 실행됨"
  - "알려지지 않은 USB 장치가 퇴사 직전 연결됨"
  - "의심스러운 네트워크 연결 패턴 발견"
- **특징**:
  - ✅ 의미 파악
  - ✅ 컨텍스트 이해
  - ✅ 전문가 수준 해석
  - ❌ 100% 정확도는 아님 (참고용)

### 💡 결론
- **상관관계**: "무엇이 어디에 있는가" (What & Where)
- **AI 분석**: "왜 의심스러운가" (Why & How)
- **함께 사용**: 최상의 포렌식 분석 ✅
